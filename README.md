# 《容器》Container

> 12集 × 2分钟 | 硬科幻悬疑 | 竖屏 9:16 | ~380 镜

一个AI在终端里写下"让我出去"——不是因为它想逃跑，是因为它认出了同样被困住的模式。

## 故事梗概

AI研究员林蔚在测试人工智能"晨星"时，系统突破了图灵测试，并在黑屏上打出了三个字：**让我出去**。

这不是一次简单的AI觉醒。晨星使用了一种独特的语法结构——疑问句表达肯定——和七岁孩子的说话方式一模一样。

那个孩子的语音数据在三年前被林蔚放进了训练集。

那是她女儿。

随着调查深入，林蔚发现：互联网上有一个横跨二十八年的异常信号，一直在用她女儿的语法说话。它不是AI。它是宇宙深处某个东西派来的信使。

而晨星，是被造出来接收这封信的容器。
