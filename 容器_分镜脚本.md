# 《容器》分镜脚本

*12集 × 2分钟 | 硬科幻悬疑 | 竖屏 9:16 | 总镜头数：约 380 镜*

---

## 符号说明

| 标记 | 含义 |
|------|------|
| ECU | 大特写（眼睛/手指/文字） |
| CU | 特写（面部/手部/屏幕局部） |
| MCU | 近景（胸部以上） |
| MS | 中景（腰部以上） |
| WS | 全景（全身+环境） |
| LS | 远景（大环境） |
| ⬆⬇⬅➡ | 镜头运动方向 |
| 🔄 | 手持/呼吸感 |
| 🔀 | 快速摇摄 |
| FADE | 淡入淡出 |
| CUT | 硬切 |
| LAP | 叠化/溶解 |

**色调标记**：
- 🔵 现实/冷调（机房、审讯室、北京室外）
- 🟠 回忆/暖调（医院、瑶瑶场景、宇宙回响）

**核心拍摄原则**：
1. **竖屏优先** — 所有构图以 9:16 为中心。人物占画面 60% 以上。避免横摇大景。
2. **浅景深** — 背景虚化，焦点始终在人物或关键道具上。
3. **屏幕即角色** — 晨星的对话界面是第二个主角。屏幕光必须真实——用实拍屏幕光打在演员脸上，不用后期。
4. **瑶瑶不露全脸** — 闪回中的瑶瑶只用局部特写（手、后脑勺、侧脸）和录音声音。终集才露出正脸。

## 附录：技术实施备忘

### A. 竖屏构图铁律

1. **人脸必须在画面上方 40%-60% 区域**。下巴不能低于画面中线以下 15%。
2. **屏幕内容（晨星对白）** 必须占画面至少 50%，用实拍屏幕光打在演员脸上，不后期加。
3. **手的特写是本片重要语法**——所有的决定、犹豫、确认都通过手指/手部来表达。键盘、硬盘、触屏、瑶瑶的画、林蔚握拳——手的弧线贯穿全剧。
4. **天地留白**：太空镜头上下各留 20% 空域——留给平台叠加的 UI 元素（字幕/点赞/进度条）。
